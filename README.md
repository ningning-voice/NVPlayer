Hello, I'm the ningning-voice who created NVPlayer. As you can see from the name of the program, I made my own music player to listen to the songs of my favorite K-pop idols, like Espa. It's still incomplete, but it doesn't have a problem using it (probably). The supported codecs are flac, mp3, ogg, and wav. The functionality is simple. When you select the music folder, you read the metadata in the music file and organize it in order of artist - release year - album - track number.
